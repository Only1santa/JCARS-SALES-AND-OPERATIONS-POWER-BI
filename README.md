# JCARS-SALES-AND-OPERATIONS-POWER-BI
Personal analysis on Power BI

In this dataset , the objective was to analyse sales trends and operations of the company by different car models and car yards available 
Besides , logistic operations and delivery status on the cars was also analysed .

Started by importing the data to aiven cloud server by creating a service for working and save for future reference.
Used DBeaver to import the data to the aiven cloud by creating a new database and importing the data in the named database.
Then connected power bi to my postgres sql data base to retrive the data set for transformation and modelling.

#Insights
